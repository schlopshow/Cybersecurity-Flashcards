# Schlop's Flashcards

This Repo contains all of my Anki Flashcards for every certification and project I am working on. I recommend when studying to adapt the flashcards as you see them so they become yours. What this means is that when you see a flashcard, add an image or sound effect to it to make it your own or change the wording in such a way that it will become more memorable and personal to yourself. If you don't do this, I am afraid these cards won't be of as much use!

## Importing Deck

1. Install anki https://apps.ankiweb.net/
2. Go to Release and download the 'flashcards.apkg' file
3. Open Anki
4. Click FIle at the top right corner
5. Import
6. Double click the flashcards.apkg file and it wil import all my cards into your anki instance.

## Structure of Decks

Normally, each deck will contain subdecks that go over each individual section of the course material. This allows you to pick and choose the topics you want to learn about with more granuality.
